# fuztask

Demo projet avec la stack Svelete - Bun - Hono => Its fun !

## MVP

L'idée est de construire une petite applicaiton permettant de simplifier le suivis des tâche.

- On peut créer des projets. On peut ensuite ajouter des tâches sur chaque projets.
- Sur une page de création de tâche, on accède à un formulaire permettant de reseigner les champs comme sur "template issue Github" définis sur le projet Ubklub.
- De cette manière, on peut faire un suvis de tâche simplifier sur cette app
